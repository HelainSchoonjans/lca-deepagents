# Quiz: Skills

> **Live lesson:** https://langchain-ai.github.io/lca-deepagents/m3/m3.2-skills-quiz.html

**Q1: What constraint links the skill directory name to its SKILL.md frontmatter?**

A) The directory name must match the `name` field in the frontmatter
B) The directory name must match the `description` field
C) The directory name must be all uppercase
D) There is no constraint; the two are independent

*Answer: A — Breaking this constraint means the SDK cannot resolve the skill correctly.*

---

**Q2: At which stage does the agent call `read_file` on a skill's SKILL.md path?**

A) Discovery: when the SDK assembles the system prompt at startup
B) Execution: once per tool call during the run
C) All three stages; the file is read once and cached at startup
D) Activation: when a task matches the skill's description

*Answer: D — At discovery, only the name and description are injected; the full body stays unread until a matching task triggers the read.*

---

**Q3: What should the `description` field in a skill's frontmatter communicate?**

A) A summary of what the skill does internally, step by step
B) The skill's author and version number
C) In what situation the skill should be used
D) The list of tools the skill is permitted to call

*Answer: C — The description is the only text visible to the agent before it decides whether to activate a skill.*

---

**Q4: Why does the SDK inject only the name and description into the system prompt rather than the full SKILL.md body?**

A) The full body is too long for any model's context window
B) To keep the agent's context lean by loading skill details only when a task requires them
C) The full body is compiled into a binary at deploy time and is not readable as text
D) Injecting the full body would expose internal instructions to users

*Answer: B — Progressive disclosure keeps the context lean.*

---

**Q5: Which two frontmatter fields in SKILL.md are required?**

A) `name` and `allowed-tools`
B) `name` and `compatibility`
C) `description` and `metadata`
D) `name` and `description`

*Answer: D — `name` must be lowercase alphanumeric with hyphens and must match the directory name.*
