# Quiz: Tools

> **Live lesson:** https://langchain-ai.github.io/lca-deepagents/m1/m1.5-tools-quiz.html

**Q1: You pass `tools=[my_tool]` to `create_deep_agent`. What happens to the built-in tools like `read_file` and `write_todos`?**

A) They are merged with `my_tool`; both are available
B) They are replaced by `my_tool`
C) They are disabled until re-added explicitly
D) They move to a separate tool list the agent accesses on request

*Answer: A — The `tools` parameter is additive. Whatever you pass is merged with the SDK's built-in tool suite.*

---

**Q2: When an LLM decides it needs to call a tool, what does it actually produce?**

A) It calls the Python function directly
B) A ToolMessage with the result already included
C) A plain text description of what it would do
D) An AIMessage with a `tool_calls` list specifying the name and arguments

*Answer: D — The LLM never executes code. The Tool Node inside the deep agent reads the `tool_calls` list and performs the actual execution.*

---

**Q3: Which part of a `@tool` definition does the model read to decide when to reach for that tool?**

A) The function name
B) The docstring
C) The type annotations on the parameters
D) The return type annotation

*Answer: B — The docstring is the description the model uses to decide when to call a tool.*

---

**Q4: Why does Human-in-the-Loop require a checkpointer?**

A) To save graph state so execution can pause and resume from the same point after a human decision
B) To log tool call arguments to a file for audit purposes
C) To validate that the human's approval matches the tool schema
D) To prevent the LLM from calling the tool more than once

*Answer: A*

---

**Q5: In the SQL lab, `write_sql` calls `interrupt()` before `db.run()`. At that moment, has the database been modified?**

A) Yes: the write runs first, then waits for approval to commit
B) It depends on the checkpointer used
C) No: `interrupt()` fires before `db.run()`, so nothing has been written yet
D) Yes: a draft write is created and rolled back on rejection

*Answer: C*
