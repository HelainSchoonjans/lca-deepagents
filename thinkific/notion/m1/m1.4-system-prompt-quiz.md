# Quiz: The System Prompt

> **Live lesson:** https://langchain-ai.github.io/lca-deepagents/m1/m1.4-system-prompt-quiz.html

**Q1: How many named parts does `create_deep_agent` assemble the system prompt from?**

A) One: the BASE prompt only
B) Two: USER and BASE
C) Two: USER and SUFFIX
D) Three: USER, BASE, and SUFFIX

*Answer: D. The assembled prompt has three parts: USER (your `system_prompt` argument), BASE (the SDK's built-in `BASE_AGENT_PROMPT`), and SUFFIX (appended by a `HarnessProfile` for model-specific tuning).*

---

**Q2: Where in the assembled prompt does the USER segment appear?**

A) At the very front, before BASE and SUFFIX
B) At the very end, after the SUFFIX
C) After the BASE instructions
D) Interleaved with the BASE instructions

*Answer: A. The SDK slots the USER segment at the very front of the assembled prompt, ahead of the BASE instructions and any SUFFIX.*

---

**Q3: You write `SYSTEM_PROMPT = 'You are a pirate.'` but the agent replies with no pirate tone. What is most likely wrong?**

A) The BASE prompt overrides all user-defined personas
B) The model requires a special flag to enable custom personas
C) SYSTEM_PROMPT was defined but not passed to `create_deep_agent`
D) The SUFFIX always overrides the USER segment

*Answer: C. Defining the string is not enough. It must be passed explicitly: `create_deep_agent(model=model, system_prompt=SYSTEM_PROMPT)`.*

---

**Q4: What is the BASE segment of the assembled prompt?**

A) The `system_prompt` string you pass to `create_deep_agent`
B) The tool schemas appended by the SDK before each LLM call
C) A model-specific tuning block provided by a HarnessProfile
D) The SDK's built-in `BASE_AGENT_PROMPT`, the behavior guidance that is always present

*Answer: D. BASE is the SDK's built-in `BASE_AGENT_PROMPT`. It contains behavior guidance that is always present, regardless of what you pass as `system_prompt`.*

---

**Q5: You switch from a pirate persona to a cowboy persona by changing `SYSTEM_PROMPT`. What else in the assembled prompt changes?**

A) The BASE instructions adapt to match the new persona
B) Nothing; only your USER segment changes
C) The SUFFIX is regenerated based on the persona
D) The tool schemas update to reflect the new persona's capabilities

*Answer: B. Only your USER segment changes. Everything else stays put.*
